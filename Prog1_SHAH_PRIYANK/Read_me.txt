-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------COMMANDS---------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

	A) CPU [threadno] [i,f]

		i.e. to run for 2 thread and integer operation,
		./Assign1Run.sh CPU 2 f
	
	B) Disk [blocksize] [threadno]	

		i.e. to run for 2 thread and 1 KB Block,
		./Assign1Run.sh Disk 1024 2
	
	C) Network:
	
		-> To run TCP program, open 2 different intance of amazon ec2.
		-> Make Sure you run your Server first than Run you Client. 
		
		1) TCP:
	
		Client:
		./Assign1Run.sh tcp-Client [threadno] [blocksize] [hostname_server]	
			i.e. ./Assign1Run.sh tcp-Client 2 65536 localhost
		Server:
		./Assign1Run.sh tcp-Server [threadno] [blocksize] 
			i.e. ./Assign1Run.sh tcp-Server 2 65536 	
		
		2) UDP:

		
		Client:
		./Assign1Run.sh udp-Client [threadno] [blocksize] [hostname_server]	
			i.e. ./Assign1Run.sh tcp-Client 2 65507 localhost
		Server:
		./Assign1Run.sh udp-Server [threadno] [blocksize] [hostname_client] 
			i.e. ./Assign1Run.sh tcp-Server 2 65507 localhost
	
		NOTE: UDP Allows transmission of maximum of 65507 Bytes only, if you try to send more than the given range then it will throw an error.
	
		in order to obtain hostname , go to amazon ec2 page and find public DNS, which looks like "ec2-52-90-237-215.compute-1.amazonaws.com", use it as a 			hostname.
